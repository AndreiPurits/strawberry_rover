## rover_simulation

ROS 2 package for virtual rover motion simulation in field rows.

### Implemented node

- `rover_pose_simulator`
  - subscribes: `/cmd_vel` (`geometry_msgs/msg/Twist`)
  - publishes: `/sim/rover_pose` (`geometry_msgs/msg/PoseStamped`)
  - publishes TF: `sim_world -> base_link`
  - simulates 2D motion state (`x`, `y`, `yaw`) at 20 Hz
  - supports `motion_mode=scripted_rows` for deterministic row-to-row traversal across 5 rows

### Motion update

- `x += v * cos(yaw) * dt`
- `y += v * sin(yaw) * dt`
- `yaw += angular_z * dt`

Where:
- `v = linear.x`
- `angular_z = angular.z`

In `scripted_rows` mode, velocities are generated by an internal waypoint state machine:
- drive along current row
- turn at row end
- align to next row
- repeat until last row

### Launch

```bash
ros2 launch rover_simulation rover_pose_simulator.launch.py
```
